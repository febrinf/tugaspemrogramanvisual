# 📖 Aplikasi Peminjaman Buku
Aplikasi ini dibuat menggunakan C# Windows Forms di Visual Studio 2022 untuk mempermudah pengguna dalam melakukan peminjaman buku.

# Fitur Aplikasi
✔ Tambah Peminjaman → Pengguna bisa memasukkan Nama Peminjam, Judul Buku, memilih Tanggal Pinjam dan Tanggal Kembali.
✔ Simpan Data ke CSV → Setiap peminjaman yang dibuat akan disimpan dalam file peminjaman.csv agar bisa dibuka kembali.
✔ Tampilkan Data Peminjaman → Data yang sudah tersimpan akan otomatis muncul di DataGridView saat aplikasi dibuka.
Menampilkan daftar peminjaman buku dengan kolom:
     - Nama Peminjam: Menampilkan nama peminjam.  
     - Judul Buku: Menampilkan judul buku yang dipinjam.  
     - Tanggal Pinjam: Menampilkan tanggal peminjaman.  
     - Tanggal Kembali: Menampilkan tanggal pengembalian. 
# Cara Menggunakan
Jalankan Aplikasi (F5 di Visual Studio 2022).
Masukkan Nama Peminjam, Judul Buku, pilih Tanggal Pinjam dan Tanggal Kembali.
Klik "Pinjam" untuk menyimpan data.
Data otomatis tersimpan di peminjaman.csv dan muncul di tabel.

Saat aplikasi dibuka kembali, semua data peminjaman sebelumnya akan muncul. Aplikasi ini memberikan kemudahan dalam mengelola peminjaman buku, serta memastikan data tersimpan dengan rapih dan mudah diakses kapan saja.
